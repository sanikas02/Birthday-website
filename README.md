# Birthday Celebration Website 🎉

This is a fun and interactive **Birthday Celebration Website** where users can enter their name and enjoy a personalized birthday greeting with animations, music, and decorations like balloons, stars, and glitter.

## Features ✨
- **Personalized Greeting**: The user's name is displayed on the celebration page.
- **Decorations**: Includes floating balloons, twinkling stars, and glitter effects.
- **Music**: Plays a birthday song in the background.
- **Responsive Design**: Works on both desktop and mobile devices.

## How to Use 🚀
1. Open the `index.html` file in a browser.
2. Enter your name in the input field and click the "Let's Celebrate!" button.
3. Enjoy the personalized birthday celebration on the next page!
